## はじめに
本リポジトリでは、**長期投資**を想定した株価予測モデル構築を行う。  
株価データは、任天堂のデータを用いる。  
任天堂のデータは、Pythonを用いて、stooqから抽出する。  
本リポジトリの詳細は、下記やコードを参照すること。

**注意**  
本リポジトリは、自己学習を目的として作成したものであるため、内容の信頼性はないことに注意する。
